# 1°Avaliação

Questoes da avaliacao com foco em logica e estruturas de repeticao.

## Arquivos de codigo
- `Pedro_Mariano_Questao1.c`: verifica se uma sequencia informada esta em ordem crescente.
- `Pedro_Mariano_Questao2.c`: processa caracteres e calcula metricas envolvendo primos e digitos.
- `Pedro_Mariano_Questao3.c`: gera sequencias de Fibonacci para valores em um intervalo.

## Como usar
- Compile e execute cada questao separadamente no terminal.
