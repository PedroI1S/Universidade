# Matrizes

Exercicios de manipulacao de matrizes em C.

## Arquivos de codigo
- `12.06_Ex3.c`: preenche matriz 10x10 e zera diagonais principal/secundaria.
- `12.06_ex2.c`: calcula soma de linhas e colunas de matriz 5x5.
- `16.06_Ex4.c`: tentativa de exercicio com matriz e potencia (incompleto).
- `Matrizes.h`: cabecalho com definicoes auxiliares para os exercicios.

## Como usar
- Compile cada arquivo para validar a matriz gerada e os resultados impressos.
