# Estrutura Sequencial

Listas de exercicios com calculos diretos e transformacoes numericas.

## Subpastas
- `Lista 1`: exercicios de extracao e reorganizacao de digitos.
- `Lista 1 - entrega`: exercicios de conversao de unidades e distribuicao de valores.
- `Lista 2`: problemas de classificacao simples e formulas aplicadas.

## Como usar
- Compile cada arquivo C separadamente.
