# P1

Total de arquivos de programacao neste ramo: 5

## O que cada codigo faz

- comb_drive_equilibrium.m: calcula numericamente o ponto de equilibrio estatico do atuador comb-drive para uma tensao constante.
- comb_drive_ode.m: define as EDOs do sistema eletromecanico (x, xdot, q, i) incluindo capacitancia variavel e forca eletrostatica.
- maldade.m: script de apoio para resolucao de exercicios de sistemas de controle (Bode, Nyquist, margens e estimativas temporais).
- Q3_comb_drive.m: script principal da simulacao do comb-drive, integra as EDOs com ode45, calcula grandezas derivadas e plota respostas.
- Script_P1.mlx: live script da P1 com desenvolvimento, calculos e interpretacao dos resultados do exercicio.
