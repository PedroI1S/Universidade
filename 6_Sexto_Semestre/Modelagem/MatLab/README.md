# MatLab

Total de arquivos de programacao neste ramo: 3

## O que cada codigo faz

- nyqst.m: monta uma funcao de transferencia e gera o diagrama de Nyquist do sistema, com ajuste de escala dos eixos.
- SimulacaoNumericaDeSistemasDinamicos.mlx: live script de simulacao numerica de sistemas dinamicos com resultados documentados no proprio arquivo.
- SistemasDinamicos.m: simula um conversor chaveado nao ideal via modelo em espaco de estados com PWM e plota a tensao no capacitor.
