# JogoDaVelha

Total de arquivos de programacao neste ramo: 2

## O que cada codigo faz

- JogoDaVelha_minmax.py: jogo da velha no terminal contra IA com algoritmo Minimax completo para escolher jogadas otimas do computador.
- tictactoe_astar.py: jogo da velha com geracao do grafo de estados e decisao por busca A* com heuristica de forks/vitorias e resposta adversarial do oponente.
