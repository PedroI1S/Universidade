# Agebra linear

Resumo dos roteiros de algebra linear desenvolvidos em MATLAB Live Script.

## Arquivos de codigo
- `Matrizes 1.mlx`: introducao a operacoes com matrizes (declaracao, acesso e manipulacoes basicas).
- `Matrizes 2.mlx`: exercicios intermediarios com combinacoes lineares e operacoes matriciais.
- `Matrizes 3.mlx`: continuidade dos exercicios de matrizes com etapas de resolucao e validacao no MATLAB.

## Como usar
- Abra os arquivos `.mlx` no MATLAB para visualizar texto, codigo e resultados integrados.
