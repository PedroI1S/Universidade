# 5 - Pilhas

Exercicios de pilha sequencial e aplicacoes praticas.

## Arquivos de codigo
- `Ex1.c`: usa pilha para converter numeros decimais para representacao octal.
- `Ex2.c`: verifica balanceamento de parenteses com pilha de caracteres.
- `Ex3.c`: tentativa de calculadora baseada em pilha para operadores e operandos.
- `Ex_nota.c`: validacao de parenteses balanceados para avaliacao.
- `teste.c`: versao de teste do verificador de parenteses com pilha.

## Como usar
- Compile cada arquivo C separadamente; os programas esperam entradas pelo terminal.
