# 3 -TAD

Implementacoes de Tipos Abstratos de Dados em C.

## Arquivos de codigo
- `Aula 1.c`: define o tipo `Real` (parte inteira + fracao) com operacoes de soma, subtracao e arredondamento.
- `Ex2.c`: implementa TAD `Matriz` com alocacao 2D e operacoes como alterar, extrair linha/coluna e submatriz.
- `ExN.c`: implementa TAD `Vetor` dinamico com insercao, intercalacao, concatenacao e subvetor.
- `Tad.c`: versao alternativa do TAD `Vetor`, usada como base de exercicio/experimento.

## Como usar
- Compile cada arquivo separadamente para testar o TAD correspondente.
