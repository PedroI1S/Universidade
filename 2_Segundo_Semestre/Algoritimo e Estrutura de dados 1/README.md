# Algoritimo e Estrutura de dados 1

Organizacao dos exercicios de C da disciplina, seguindo a progressao de conceitos basicos para TADs e estruturas lineares.

## Subpastas
- `1 - Ponteiros e Structs`: manipulacao de ponteiros, vetores de structs e processamento de registros.
- `2 - Alocacao dinamica`: uso de `malloc`/`realloc` e vetores dinamicos.
- `3 -TAD`: implementacoes de tipos abstratos como `Real`, `Matriz` e `Vetor`.
- `4 - Listas estaticas`: listas baseadas em vetor com insercao, remocao e busca.
- `5 - Pilhas`: pilha sequencial aplicada a conversao de base e balanceamento de parenteses.
- `6 - Filas`: filas sequenciais/circulares e exercicios de intercalacao e simulacao.

## Como usar
- Compile os arquivos C individualmente para testar cada exercicio.
- Consulte os `README.md` internos para o papel de cada arquivo.
