# 1 - Ponteiros e Structs

Exercicios de structs, vetores de registros e manipulacao de dados em C.

## Arquivos de codigo
- `Aula 1.c`: cadastra alunos em uma struct de boletim, calcula medias e destaca melhor/pior desempenho.
- `Aula_2.c`: arquivo de aula sem implementacao efetiva de logica (rascunho).
- `Aula_3.c`: calcula ocorrencia e percentual de um caractere dentro de uma string.
- `Ex notas.c`: consolida estatisticas de jogadores de volei e calcula percentuais de acerto por fundamento.
