# Lista_DEncadeada

Implementacoes de lista duplamente encadeada.

## Arquivos de codigo
- `ListaDE.c`: TAD base da lista duplamente encadeada com operacoes fundamentais.
- `ListaDE.h`: declaracoes de tipos e funcoes da lista.
- `ListaDE_RemocaoRepeticao.c`: variacao com rotina para eliminar elementos repetidos.

## Como usar
- Compile com foco em testes de insercao, remocao e percurso bidirecional.
