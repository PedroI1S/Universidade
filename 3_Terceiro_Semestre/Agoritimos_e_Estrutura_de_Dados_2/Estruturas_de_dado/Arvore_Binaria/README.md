# Arvore_Binaria

Exercicios sobre Arvore Binaria de Busca (ABB).

## Arquivos de codigo
- `EspelharABB.c`: espelha os ramos de uma ABB.
- `ImprimirFolhas.c`: percorre a ABB imprimindo apenas os nos folha.
- `Inserir_void.c`: exemplo de insercao com ponteiro duplo para alterar a raiz.
- `Somatorio_nos.c`: calcula a soma dos valores armazenados nos nos.
- `TAD_ABB.c`: implementacao completa da ABB com inserir, buscar e remover.

## Como usar
- Compile os arquivos individualmente para testar cada operacao.
