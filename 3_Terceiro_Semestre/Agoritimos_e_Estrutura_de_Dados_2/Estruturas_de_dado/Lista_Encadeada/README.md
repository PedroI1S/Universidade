# Lista_Encadeada

Exercicios de lista simplesmente encadeada.

## Arquivos de codigo
- `OrdenacaoDeString.c`: ordena strings armazenadas em lista encadeada.
- `ProcurarSublista.c`: procura uma sublista dentro de outra lista encadeada.
- `TAD.c`: implementacao base do TAD de lista simplesmente encadeada.

## Como usar
- Compile separadamente para validar cada estrategia de manipulacao da lista.
